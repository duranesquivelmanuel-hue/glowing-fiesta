# glowing-fiesta
This project is for everyone who likes having music playlists on the web; the first step involves a small connection with a button and an array for the music.

Use Audio player using the Web Audio API to load and play multiple samples.

## Current Features
- Load multiple audio files
- Play samples with the Web Audio API
- Basic playback control

## Technologies
- JavaScript (Vanilla)
- Web Audio API
- HTML5 / CSS3 basic
